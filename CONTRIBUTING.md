# Contributing

Thank you for considering contributing to the TypeScript Programming Playground! This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md).

## How to Contribute

1. **Fork and clone** the repository
2. **Create a feature branch** with `git checkout -b feature/your-feature`
3. **Make your changes** and follow conventional commits
4. **Test thoroughly** - ensure all examples run correctly
5. **Submit a pull request** with clear description

## Code Style

- Use TypeScript for all new code
- Follow existing conventions in the codebase
- Test all code examples before submitting
- Update documentation where necessary

## Good First Issues

Look for issues marked `good first issue` or `help wanted`.

## Pull Request Process

1. Ensure tests pass locally
2. Write a clear PR description
3. Reference related issues in the PR body
4. Request review from maintainers

## Development Setup

See `extension/QUICKSTART.md` for detailed instructions on running the extension locally.
